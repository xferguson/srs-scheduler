SRS Scheduler

This is a tool designed to show you how many flash card reviews you can expect per day using various Spaced Repetition Systems. The SRS algorithms were taken from the Memrise website message boards and the Clozemaster website. To see the code, or to download so you can use the command line tool, please check out the github repo.

The current graph is calculated using 20 new flash cards per day over the course of 90 days. You can edit the Daily New Cards and the Total Days with the inputs below the graph.
